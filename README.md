## 프로젝트 설치
`node v18.15.0` 환경에서 개발되었습니다. nodejs를 설치해주세요.
https://nodejs.org/ko

### 1. 프로젝트 클론
```bash
git clone <해당 프로젝트 url>
```

### 2. 프로젝트 의존성 설치
```bash
npm install
```

### 3. 프로젝트 실행
```bash
npm run start
```
